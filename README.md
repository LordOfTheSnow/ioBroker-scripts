# ioBroker-scripts
Collection of Blockly/JavaScript script for ioBroker

## Contents

### fritzbox-call-on-lg-tv

A script for ioBroker to display an incoming phone call via the Fritzbox TR-064 protocol as a popup on a LG webOS TV.
